## 项目思路

数据合并是见在经过自动化打标工具获取到的VOC数据集，将一个视频中的多个标签物体合并，从而提升多目标的检测性能；自动化数据增强则是通过对数据集图像进行旋转、裁剪、改变曝光度等，提升整体数据集图片的数量，从而提升训练的效果。
