# 自动化打标工具
## 项目思路
5min从视频到边缘设备检测目标物体落地，项目通过SAM大模型可实现提取目标检测物体，并可自动追踪并获取VOC等数据格式，极大的减低了人工打标签的成本，实现嵌入式视觉识别的快速低成本落地，为机器人感知环境提供一套完整的解决方案。
